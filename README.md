# IBVS Ship Tracking — Observability Analysis

MATLAB simulation for Image-Based Visual Servoing (IBVS) 
applied to ship tracking using a UAV-mounted camera.

## Files
- main.m — run this to execute the full simulation
- config.m — all simulation parameters
- ship_motion.m — CV and CA motion models
- camera_projection.m — 3D to 2D projection
- observability_analysis.m — rank and condition number analysis

## How to Run
Open MATLAB, navigate to this folder, and run: main
